Windows Setup Guide

SKD International — Development Environment Setup for Windows 10/11

1 Windows Terminal & Package Manager

Install Windows Terminal from the Microsoft Store, then set up winget (built into Windows 11) or install Scoop.

Open all terminals as Administrator for installation steps.
# Option A: winget (built-in on Windows 11)
winget --version

# Option B: Scoop (if winget unavailable)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression

2 Core Tools

Install Git, Python, Node.js, and Docker.

# Using winget
winget install Git.Git
winget install Python.Python.3.12
winget install OpenJS.NodeJS.LTS
winget install Docker.DockerDesktop
winget install GitHub.cli

# Restart terminal after installations

# Verify
git --version
python --version
node --version
docker --version
gh --version

3 SSH Key & GitHub

Generate an SSH key and connect to GitHub.

# Generate SSH key
ssh-keygen -t ed25519 -C "your-email@example.com"

# Start SSH agent
Get-Service ssh-agent | Set-Service -StartupType Automatic
Start-Service ssh-agent
ssh-add ~/.ssh/id_ed25519

# Copy public key to clipboard
Get-Content ~/.ssh/id_ed25519.pub | Set-Clipboard

# Add to GitHub: Settings → SSH Keys → New SSH Key → Paste

# Authenticate with GitHub CLI
gh auth login

# Clone the org repos
mkdir ~/Documents/skd
cd ~/Documents/skd
gh repo clone SKD-international/brain
gh repo clone SKD-international/trading-infra
gh repo clone SKD-international/oanda-mcp
gh repo clone SKD-international/docs

5 Python Environment

Set up Python with a virtual environment for trading and ML work.

# Create project venv
cd ~/Documents/skd/trading-infra
python -m venv .venv
.venv\Scripts\activate

# Install core packages
pip install fastapi uvicorn redis aiosqlite httpx
pip install catboost lightgbm xgboost scikit-learn
pip install pandas numpy
pip install oandapyV20
Note: TA-Lib on Windows requires a pre-built wheel. Download from github.com/cgohlke/talib-build and install with pip install TA_Lib‑*.whl

6 Obsidian (BrainVault)

Set up the shared knowledge base with Git sync.

# The brain repo is already cloned from Step 3
# Open Obsidian → Open Folder as Vault → ~/Documents/skd/brain

# Install community plugins:
# 1. Settings → Community Plugins → Turn on
# 2. Browse → search "Obsidian Git" → Install → Enable
# 3. Obsidian Git settings:
#    - Auto pull interval: 5 minutes
#    - Auto push interval: 10 minutes
#    - Pull on startup: ON

7 Claude Code

Install Claude Code for AI-powered development. optional

# Install Claude Code
npm install -g @anthropic-ai/claude-code

# Run in any project directory
cd ~/Documents/skd/trading-infra
claude

8 Hetzner VPS Access

Connect to the shared production server.

# Send your SSH public key to Niksa
Get-Content ~/.ssh/id_ed25519.pub

# Once added, connect:
ssh your-username@204.168.169.124

# Services running:
# - n8n: http://204.168.169.124:5678
# - Netdata: http://204.168.169.124:19999

+ WSL2 (Advanced)

For a Linux-like development experience on Windows. optional

# Install WSL2 with Ubuntu
wsl --install

# After restart, set up Ubuntu:
sudo apt update && sudo apt upgrade -y
sudo apt install python3 python3-pip python3-venv nodejs npm redis-tools

# Docker Desktop integrates with WSL2 automatically
WSL2 gives you a real Linux environment. Recommended if you want parity with the Hetzner server (Ubuntu ARM64).
