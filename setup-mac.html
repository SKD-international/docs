Mac Setup Guide

SKD International — Development Environment Setup for macOS

1 Homebrew & CLI Tools

Homebrew is the package manager for macOS. Everything else installs through it.

# Install Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Essential CLI tools
brew install git python@3.12 node redis sqlite gh
brew install --cask docker

2 SSH Key & GitHub

Generate an SSH key and add it to your GitHub account.

# Generate SSH key
ssh-keygen -t ed25519 -C "your-email@example.com"

# Copy public key to clipboard
pbcopy < ~/.ssh/id_ed25519.pub

# Add to GitHub: Settings → SSH Keys → New SSH Key → Paste

# Authenticate with GitHub CLI
gh auth login

# Clone the org repos
mkdir -p ~/Documents/skd
cd ~/Documents/skd
gh repo clone SKD-international/brain
gh repo clone SKD-international/trading-infra
gh repo clone SKD-international/oanda-mcp
gh repo clone SKD-international/docs

4 Python Environment

Set up Python with a virtual environment for trading and ML work.

# Create project venv
cd ~/Documents/skd/trading-infra
python3 -m venv .venv
source .venv/bin/activate

# Install core packages
pip install fastapi uvicorn redis aiosqlite httpx
pip install catboost lightgbm xgboost scikit-learn
pip install pandas numpy ta-lib
pip install oandapyV20

5 Obsidian (BrainVault)

Set up the shared knowledge base with Git sync.

# The brain repo is already cloned from Step 2
# Open Obsidian → Open Folder as Vault → ~/Documents/skd/brain

# Install community plugins:
# 1. Settings → Community Plugins → Turn on
# 2. Browse → search "Obsidian Git" → Install → Enable
# 3. Obsidian Git settings:
#    - Auto pull interval: 5 minutes
#    - Auto push interval: 10 minutes
#    - Pull on startup: ON

6 Claude Code

Install Claude Code for AI-powered development. optional

# Install Claude Code
npm install -g @anthropic-ai/claude-code

# Run in any project directory
cd ~/Documents/skd/trading-infra
claude

7 Hetzner VPS Access

Connect to the shared production server.

# Send your SSH public key to Niksa
cat ~/.ssh/id_ed25519.pub

# Once added, connect:
ssh your-username@204.168.169.124

# Services running:
# - n8n: http://204.168.169.124:5678
# - Netdata: http://204.168.169.124:19999
